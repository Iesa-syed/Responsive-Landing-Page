# Responsive Landing Page

A modern and responsive landing page built using pure HTML and CSS.

## Features

- Fixed Navigation Bar
- Fullscreen Hero Section
- Background Image Overlay
- Modern Typography
- Responsive Layout
- Smooth Hover Effects
- Call-To-Action Buttons
- Clean UI Design

## Technologies Used

- HTML
- CSS
- Google Fonts (Poppins)

## Project Structure

├── index.html
├── style.css
├── simple-website-bg-img.jpg
└── README.md
